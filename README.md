This is a comprehensive screenplay/script editing and analysis application built with vanilla JavaScript. It enables users to write, format, and analyze dramatic texts (screenplays, plays, etc.) with intelligent character recognition and dialogue analysis features.

## Key Features

- **Text formatting**: Bold, Italic, Underline support
- **Character detection**: Automatic recognition of character names and roles
- **Analysis tools**: Word counting, line counting per character, misspelling detection, character grouping by scene
- **Script insights**: Full character scripts with previous/next dialogue context, scene detection, character interaction analysis
